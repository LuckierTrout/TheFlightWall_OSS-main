Project-context scope rules exclude pre-existing issues from the reviewed story's verdict. Carry forward as a fn-cleanup candidate.

- **Claimed Issue**: Touch targets fail WCAG 44×44px because `.wizard-test-actions .nav-btn` only sets `min-height:44px` not `min-width:44px` | **Raised by**: Reviewer D, B | **Dismissal Reason**: The buttons use `width:100%` (inherited from `.wizard-test-actions .nav-btn { width:100% }`), so rendered width is always the full card width (minimum 320px on the minimum supported viewport).
