## Changes Applied

**File**: `firmware/core/ConfigManager.cpp`
**Change**: Add `_callbacks.clear()` to `init()` to prevent stale callback UB across repeated `init()` calls in tests.
**Before**:
```cpp
    {
        ConfigLockGuard guard;
        _persistPending = false;
        _persistScheduledAt = 0;
        loadDefaults();
    }
```
**After**:
```cpp
    {
        ConfigLockGuard guard;
        _persistPending = false;
        _persistScheduledAt = 0;
        _callbacks.clear();  // Prevent stale callbacks from prior init() calls (e.g., repeated test setUp)
        loadDefaults();
    }
```

---

**File**: `firmware/core/ConfigManager.cpp`
**Change**: Track NVS clamping with `anyCorrupted` flag; call `schedulePersist(0)` when any schedule field was clamped so corrected values are written back to NVS on the first loop tick.
**Before**:
```cpp
    // Validate schedule numeric fields on NVS load...
    if (prefs.isKey("sched_enabled")) {
        uint8_t v = prefs.getUChar("sched_enabled", snapshot.schedule.sched_enabled);
        if (v > 1) { LOG_W("ConfigManager", "Corrupted sched_enabled in NVS — reset to 0"); v = 0; }
        ...
    // [remaining clamp blocks without anyCorrupted tracking]
    prefs.end();
    { ConfigLockGuard guard; ... }
    LOG_I("ConfigManager", "NVS values loaded");
```
**After**:
```cpp
    bool anyCorrupted = false;
    if (prefs.isKey("sched_enabled")) {
        uint8_t v = prefs.getUChar("sched_enabled", snapshot.schedule.sched_enabled);
        if (v > 1) { LOG_W(...); v = 0; anyCorrupted = true; }
        ...
    // [remaining clamp blocks set anyCorrupted = true]
    prefs.end();
    { ConfigLockGuard guard; ... }
    if (anyCorrupted) {
        schedulePersist(0);
        LOG_W("ConfigManager", "Scheduling NVS self-heal persist for corrupted schedule fields");
    }
    LOG_I("ConfigManager", "NVS values loaded");
```
