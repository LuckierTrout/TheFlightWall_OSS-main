## Changes Applied

**File**: `firmware/data-src/editor.js`
**Change**: Added `dirty = true;` inside the `!found` coercion guard in `showPropsPanel` so that opening a widget with a stale/invalid content key marks the layout as needing a save, matching the in-memory state that was already being synced.

**Before**:
```javascript
        if (!found && selectedIndex >= 0 && widgets[selectedIndex]) {
          widgets[selectedIndex].content = coerced;
        }
```
**After**:
```javascript
        if (!found && selectedIndex >= 0 && widgets[selectedIndex]) {
          widgets[selectedIndex].content = coerced;
          dirty = true;
        }
```

**File**: `firmware/data/editor.js.gz`
**Change**: Regenerated via `gzip -9 -c firmware/data-src/editor.js > firmware/data/editor.js.gz`. New size: 6,577 bytes.
