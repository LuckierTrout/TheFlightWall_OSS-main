**Change**: Sync `widgets[selectedIndex].content` when the select falls back to `meta.contentOptions[0]` due to a stale/invalid content value (programmatic `.value = x` does not fire `change` event)

**Before**:
```javascript
        contentSelect.value = found ? desired : meta.contentOptions[0];
      } else if (contentInput) {
```

**After**:
```javascript
        var coerced = found ? desired : meta.contentOptions[0];
        contentSelect.value = coerced;
        /* Sync in-memory widget if the content value was coerced to a valid
           option (stale/invalid keys from older layouts). Without this, save
           would persist the invalid key even though the UI shows the coerced
           value (programmatic .value = x does not fire 'change'). */
        if (!found && selectedIndex >= 0 && widgets[selectedIndex]) {
          widgets[selectedIndex].content = coerced;
        }
      } else if (contentInput) {
```
