**File:** `firmware/core/ConfigManager.cpp`  
**Lines:** 159-161

Inconsistent validation pattern:

- `sched_dim_end`: explicit check `if (v > 1439) return false;`

```cpp
if (strcmp(key, "sched_dim_brt") == 0) {

}
```
