## Changes Applied

**File**: `firmware/adapters/WebPortal.cpp`
**Change**: Split `_handlePutLayout` id validation into two distinct checks with clear error messages for missing vs mismatched `id` field.

**Before**:
```cpp
const char* docId = doc["id"] | (const char*)nullptr;
if (docId == nullptr || strcmp(docId, id.c_str()) != 0) {
    _sendJsonError(request, 400, "Body id does not match URL id", "LAYOUT_INVALID");
    return;
}
```
**After**:
```cpp
const char* docId = doc["id"] | (const char*)nullptr;
if (docId == nullptr) {
    _sendJsonError(request, 400, "Missing id field in layout body", "LAYOUT_INVALID");
    return;
}
if (strcmp(docId, id.c_str()) != 0) {
    _sendJsonError(request, 400, "Body id does not match URL id", "LAYOUT_INVALID");
    return;
}
```
