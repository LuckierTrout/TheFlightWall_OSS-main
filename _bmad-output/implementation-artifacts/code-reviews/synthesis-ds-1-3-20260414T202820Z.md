## Changes Applied

**File**: `firmware/core/ModeRegistry.cpp`
**Change**: Fix `init()` return-value check in factory-null and init-failure restore paths (HIGH)
**Before**:
```cpp
_activeMode = prevEntry->factory();
if (_activeMode != nullptr) {
    _activeMode->init(ctx);
}

```
**After**:
```cpp
_activeMode = prevEntry->factory();
if (_activeMode != nullptr) {
    if (!_activeMode->init(ctx)) {
        delete _activeMode;
        _activeMode = nullptr;
        _activeModeIndex = MODE_INDEX_NONE;
    } else {
        _activeModeIndex = prevIndex;
    }
} else {
    _activeModeIndex = MODE_INDEX_NONE;
}
```
