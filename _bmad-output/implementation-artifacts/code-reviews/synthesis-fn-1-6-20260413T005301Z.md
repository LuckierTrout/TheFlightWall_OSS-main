## Issues Dismissed

- **Claimed Issue**: `_sendJsonError` should replace inline JSON in ack-rollback handler | **Raised by**: Reviewer A | **Dismissal Reason**: `_sendJsonError` is a `WebPortal` class member function. The `/api/ota/ack-rollback` handler is a `[]` lambda (no `this` capture) — calling `_sendJsonError` is syntactically impossible without restructuring the handler. Inline JSON strings are the correct pattern for captureless lambdas.

- **Claimed Issue**: `OTA_MAX_SIZE` should be driven by API (`/api/status.ota_max_size`) | **Raised by**: Reviewer A | **Dismissal Reason**: Over-engineering. The 1.5MB partition size is a compile-time constant (custom_partitions.csv) that cannot change at runtime. Adding an API field and HTTP round-trip for this is disproportionate; the constant is clearly documented with its hex value.

- **Claimed Issue**: Drag-over border uses `--primary` instead of `--accent` (AC #3) | **Raised by**: Reviewer A | **Dismissal Reason**: `--accent` does not exist in the project's CSS design system (`:root` defines `--primary`, `--primary-hover`, `--error`, `--success`, `--warning` — no `--accent`). - **Claimed Issue**: Race condition in `startRebootPolling` — timeout and success could fire simultaneously | **Raised by**: Reviewer A | **Dismissal Reason**: Already fixed in Pass 1 synthesis.
